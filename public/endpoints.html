How to Pay

All paid endpoints use the x402 protocol. Send a POST — if payment is required, you'll get an HTTP 402 response with USDC payment instructions.

Automatic payment with @x402/fetch or @x402/axios. Or manually sign a USDC transfer on Base and include the proof in X-402-Payment header.

Strategos endpoints also accept Authorization: Bearer stg_... API keys as an alternative to x402.
